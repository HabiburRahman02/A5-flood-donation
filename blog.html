What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a fundamental concept in web development, DOM key concepts node, tree structure,Interactivity etc. Through the DOM, JavaScript can dynamically change the content, structure, and styles of a webpage without requiring a page reload.

How do you select an element from the DOM?

We are various ways to get the element like, getElementById, getElementsByClassName, getElementsByTagName, querySelector, querySelectorAll

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the context of the DOM is a technique where a single event listener is added to a parent element to handle events triggered by its child elements. Instead of attaching separate event listeners to each child, the parent listens for events, and through event propagation (bubbling)

How do you manipulate an element's attributes and styles using the DOM?

We manipulate an HTML element's attributes and styles in multiple ways, such as using getAttribute, setAttribute, changing innerText, adding classes with classList, and more.
